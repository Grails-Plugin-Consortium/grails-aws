Grails AWS Plugin
===========
[![Build Status](https://travis-ci.org/grails-aws/grails-aws.png?branch=master)](https://travis-ci.org/grails-aws/grails-aws)

## Getting Started

Include the plugin in the `BuildConfig.groovy`

        plugins {
            compile ':aws:1.2.12.3'
        }
